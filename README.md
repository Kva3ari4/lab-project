# ИИС ППР — Интеллектуальная информационная система поддержки принятия решений

Система для подбора и распределения молодых специалистов на программы стажировок и практик на основе анкетных данных (дипломный проект, Модуль 1–2).

## Требования

- PHP 7.4+ (с расширениями PDO, pdo_mysql, session, json)
- MySQL 5.7+ / MariaDB (доступ через phpMyAdmin)
- Веб-сервер (Apache/Nginx) с корнем в каталоге проекта или настройка виртуального хоста

## Установка

### 1. База данных

1. Откройте phpMyAdmin и создайте базу (или импортируйте схему).
2. Импортируйте файл `schema.sql` — будут созданы база, таблицы и начальные данные (роли, языки, навыки, пользователь администратора).
3. Установите пароль администратора (по умолчанию в схеме — тестовый хеш). Выполните в корне проекта:
   ```bash
   php update_admin_password.php
   ```
   После этого вход: **admin@iis.local** / **admin123**.

### 2. Конфигурация

В файле `config.php` укажите параметры подключения к MySQL:

```php
'database' => [
    'host' => 'localhost',
    'name' => 'iis_ppr',
    'user' => 'root',
    'password' => '',
    'charset' => 'utf8mb4',
],
```

### 3. Запуск

**Вариант A — встроенный сервер PHP (если PHP в PATH):**
```bash
php -S localhost:8000
```

**Вариант B — Windows, если команда `php` не найдена:**
- Запустите **`run-server.bat`** (двойной щелчок) — скрипт попытается найти PHP в XAMPP/Laragon и запустить сервер.
- Либо откройте PowerShell в папке проекта и выполните:  
  `powershell -ExecutionPolicy Bypass -File run-server.ps1`

**Вариант C — через XAMPP / OpenServer / Laragon:**
- Скопируйте проект в каталог сервера (например, `htdocs`) и настройте виртуальный хост так, чтобы **корень сайта** указывал на папку проекта.

После запуска откройте в браузере: **http://localhost:8000**

## Роли и функционал

| Роль | Возможности |
|------|-------------|
| **Студент** | Регистрация, заполнение/редактирование анкеты, отправка на рассмотрение, просмотр статуса заявки и рекомендаций по программам. |
| **HR-оператор** | Просмотр и редактирование анкет кандидатов, импорт анкет (CSV), просмотр рейтингов. |
| **Руководитель программы** | Управление программами стажировок (требования, квоты), запуск интеллектуального анализа (скоринг), просмотр рейтинга, добавление кандидатов в распределение, утверждение/отклонение распределения. |
| **Администратор** | Всё выше + управление пользователями, назначение ролей, справочник навыков. |

## Соответствие документу (Модуль 1–2)

- **F1** — Управление анкетными данными: анкеты кандидатов, редактирование, импорт CSV.
- **F2** — Управление требованиями программ: программы, квоты, минимальные требования.
- **F3–F4** — Предобработка и анализ текстов: в текущей версии упрощённый скоринг на PHP (GPA, опыт, проекты, наличие текстов); интеграция с Python/NLP — отдельный этап.
- **F5–F6** — Интеллектуальная оценка и ранжирование: запуск анализа по программе, рейтинг с обоснованием.
- **F7–F8** — Поддержка распределения и принятия решений: предложения по распределению, утверждение/отклонение.
- **F9** — Отчётность: дашборд со счётчиками, рейтинги по программам.
- **F10** — Управление доступом: роли, назначение ролей пользователям.

База данных соответствует физической модели из документа (users, roles, candidates, programs, program_quota, scoring_runs, scores, assignments, notifications, audit_log, справочники skills, languages).
